# Interview Prep — Job Search Strategy (Canada)

## Target Job Titles
```
Entry-friendly:
├── Junior Cloud Engineer
├── Cloud Support Engineer
├── Infrastructure Engineer
├── DevOps Engineer (Junior/Intermediate)
└── Platform Engineer

Search keywords on LinkedIn:
"AWS" AND ("Terraform" OR "Cloud Engineer" OR "DevOps")
Location: Canada (Remote) or your city
```

## Where to Apply
```
Direct hire:
├── Banks: TD, RBC, BMO, Scotiabank, CIBC
├── Tech: Shopify, Wealthsimple, Clio, Hootsuite
├── Telecom: Rogers, Bell, Telus
└── Government: federal/provincial IT contracts

Contract/staffing (easier to break in):
├── Randstad
├── Robert Half
├── TEKsystems
├── Hays
└── Procom
```

## LinkedIn Optimization
```
Headline: "AWS Cloud Engineer | Terraform | Docker | CI/CD | SAA Certified"

About section should mention:
- AWS certifications (CCP + SAA)
- Hands-on projects with links to GitHub
- QA background (reliability mindset)
- Canadian work authorization

Post weekly about:
- What you learned this week
- A project you built (with architecture diagram)
- A problem you solved
```

- [ ] LinkedIn profile optimized
