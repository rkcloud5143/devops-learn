# Networking — Common Ports (Memorize These)

```
┌──────┬─────────────┬──────────────────────────┐
│ Port │ Protocol    │ What it's for            │
├──────┼─────────────┼──────────────────────────┤
│ 22   │ SSH         │ Remote server access     │
│ 80   │ HTTP        │ Web traffic (unencrypted)│
│ 443  │ HTTPS       │ Web traffic (encrypted)  │
│ 3306 │ MySQL       │ Database                 │
│ 5432 │ PostgreSQL  │ Database                 │
│ 6379 │ Redis       │ Cache                    │
│ 3000 │ Node.js     │ Dev server (common)      │
│ 8080 │ Alt HTTP    │ Proxy / dev server       │
└──────┴─────────────┴──────────────────────────┘
```
