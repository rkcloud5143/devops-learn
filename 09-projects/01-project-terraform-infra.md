## What to Build
```
Terraform code that creates:
в”ңв”Җв”Җ VPC with 2 public + 2 private subnets
в”ңв”Җв”Җ Internet Gateway + NAT Gateway
в”ңв”Җв”Җ Security Groups (ALB, EC2, RDS)
в”ңв”Җв”Җ ALB with target group + health checks
в”ңв”Җв”Җ Auto Scaling Group (min=2, max=4)
в”ңв”Җв”Җ Launch Template with user data (install app)
в”ңв”Җв”Җ RDS PostgreSQL (Multi-AZ) in private subnets
в”ңв”Җв”Җ S3 bucket for Terraform remote state
в”ңв”Җв”Җ CloudWatch alarms (CPU, 5xx errors)
в””в”Җв”Җ SNS topic for alert notifications
```
